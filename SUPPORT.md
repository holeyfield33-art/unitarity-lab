# Support

## Getting help

- For usage questions and bug reports, open a GitHub Issue.
- For security concerns, follow `SECURITY.md`.

## Before opening an issue

Please include:

- Python version
- OS and hardware
- Command used
- Full error output
- Minimal reproduction steps

## Scope of support

This project is currently alpha software.

- Best effort support is provided through Issues and Pull Requests.
- APIs may change between releases.
- Research features may evolve rapidly.
